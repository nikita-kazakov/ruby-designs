EM vs REM Sizing

em units are a font sizing technique and it's in relationship to the parent container. Generally, 1 em is 16px because 16px is the standard font size of the body.

Lesson Learned: Don't use em for font sizes. You'll get lost in the cascading effects.

REM is based on (root em). It fixes the em cascading problem. It's based on html font-sizing. So all you need to do is:

html { font-size: 16px }

You can now use rems and they will be based off the font size 16 on your entire site! You can use REM units for fonts.

A good place to use ems is for padding buttons. Then the padding will increase based on the text size of the text in the button.

It's easy to build small, medium, and large buttons this way. Or you could manually scale the padding and margins as you need.

Use ems when you use media queries. It's beautiful..
